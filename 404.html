Revenge of the Octocat

This was my project for the first term of AP Computer Science (Junior year of high school). Written in Java Swing, this is a top-down shooter game where the main character is the Github mascot: the Octocat!

The story: The Github servers have been bugged up! Play as the Octocat, arm yourself with lots of \xDE\xAD\xBE\xEF to shoot into those pesky bugs and make them all go away!

Features:

  1. Boss levels on stages 11, 21, and 31
  2. Background music by Waterflame89
  3. Lots of Github references!
  4. Boss AI
